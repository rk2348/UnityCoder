Unity Coding Contest 001 – Debug.Log で文字列出力

プログラミング学習の最初の一歩である「 Hello World 」の出力に挑戦しましょう。Unityのコンソールに、固定の文字列"Hello World"を表示するためのスクリプトを作成してください。

条件

  • Unityのコンソールに出力するために Debug.Log() メソッドを使用すること
  • 出力する文字列は "Hello World" の固定文字列とすること
  • その他の変数や複雑な処理は使用しないこと

入力例

この問題では、特別な入力や変数は使用しません。

出力例（Unity コンソール）

Hello World

解答用テンプレート（C# / Unity形式）

using UnityEngine; public class BasicLogger : MonoBehaviour { void Start() { ★ここに、コンソールに "Hello World" と出力するコードを書いてください } }
