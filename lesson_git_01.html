第1回: Gitの仕組みとインストール

公開日: 2025.12.06

1. はじめに：なぜGitが必要なのか

Unityでゲームを作っていると、「間違えてコードを消してしまった」「昨日の状態に戻したい」「新しい機能を試したらエラーだらけになった」といったトラブルに必ず遭遇します。
Git（ギット）を使えば、これらを解決できます。ゲームのセーブポイントを作るように、いつでも過去の状態に戻れるようになります。

【画像: Gitの概念図（セーブポイントが並んでいるイメージ）】
  • Git: 自分のパソコン内で履歴を管理するツール
  • GitHub: Gitのデータをネット上に保存・共有するサービス（Google Driveのコード版のようなもの）

2. GitHub Desktopのインストール

Gitは通常、黒い画面（コマンドライン）で操作しますが、初心者は操作が直感的でわかりやすい公式ツール「GitHub Desktop」を使うのがおすすめです。

手順1: 公式サイトへアクセス

以下のリンクからGitHub Desktopのダウンロードページへ移動します。

GitHub Desktop 公式サイト (desktop.github.com)

【画像: 公式サイトの「Download for Windows」ボタン】

手順2: インストーラーの実行

ダウンロードしたファイル（Setupファイル）をダブルクリックして実行します。インストールは自動的に進みます。

3. GitHubアカウントの作成とログイン

GitHub Desktopが起動すると、GitHubアカウントへのログインを求められます。

【画像: 初回起動時の「Sign in to GitHub.com」画面】
  1. 「Sign in to GitHub.com」をクリックします。
  2. ブラウザが開き、GitHubのログイン画面が表示されます。
    • アカウントを持っていない場合は「Create an account」から作成してください（無料）。
  3. ログイン後、「GitHub Desktopを開きますか？」というポップアップが出るので「開く」を選択します。
  4. Configure Git（名前とメールアドレスの設定）が表示されたら、そのまま「Finish」をクリックします。
完了:
「Let's get started!」という画面が表示されれば準備完了です。次回は、実際にUnityプロジェクトを管理下に置く設定を行います。
